# NeuroSynth v5 — Data Sources (Appendix B)

All 11 real data sources that feed `data/real_v5.parquet`. Sources are grouped by
access tier. No credentials required for Tier 1.

---

## Tier 1 — Auto-download, no credentials

These sources are downloaded automatically by the CI pipeline and by
`scripts/data/v5/download_kaggle.py`, `download_physionet.py`, and `download_uci.py`.

| Dataset | Source URL | Rows | Key Features | License |
| --- | --- | --- | --- | --- |
| Alzheimer's Disease Dataset (rabieelkharoua) | `kaggle.com/datasets/rabieelkharoua/alzheimers-disease-dataset` | 2,149 | MMSE, CDR, APOE4, hippocampus, ADL, depression | CC BY 4.0 |
| Dementia Prediction (shashwatwork) | `kaggle.com/datasets/shashwatwork/dementia-prediction-dataset` | 373 | OASIS-2 tabular: eTIV, nWBV, ASF, MMSE, CDR | Open |
| UCI Parkinson's Telemonitoring | `archive.ics.uci.edu/dataset/189/parkinsons+telemonitoring` | 5,875 | 22 acoustic voice features, UPDRS motor/total | Open |
| UCI Parkinson's Classic | `archive.ics.uci.edu/dataset/174/parkinsons` | 195 | 22 biomedical voice measurements, binary status | Open |
| PhysioNet PADS (smartwatch) | `physionet.org/content/parkinsons-disease-smartwatch` | 1,044 | Actigraphy, tremor, gait, HR, wrist motion | Open |
| PhysioNet Non-EEG Neurological | `physionet.org/content/noneeg-neurological-status` | 2,512 | EDA, temperature, acceleration, HR, SpO2 | Open |
| PhysioNet COVID-19 + MS | `physionet.org/content/coronavirus-nanopore` | 347 | Demographics, comorbidities, MS diagnosis | Open |
| OpenNeuro BIDS (clinical sidecars) | `openneuro.org` via `openneuro-py` | 1,444 | `participants.tsv` clinical metadata | CC0 |

**Kaggle download** — requires `~/.kaggle/kaggle.json` or env vars `KAGGLE_USERNAME` / `KAGGLE_KEY`:
```sh
pip install kaggle
kaggle datasets download rabieelkharoua/alzheimers-disease-dataset -p data/raw/
kaggle datasets download shashwatwork/dementia-prediction-dataset -p data/raw/
```

**PhysioNet download** — open datasets, no account needed:
```sh
pip install wfdb
python scripts/data/v5/download_physionet.py
```

**UCI download** — direct HTTP, no account needed:
```sh
python scripts/data/v5/download_uci.py
```

**OpenNeuro download** — public BIDS datasets:
```sh
pip install openneuro-py
python scripts/data/v5/scrape_openneuro.py --max-datasets 20
```

## Tier 2 — Free registration, high value

These require a signed Data Use Agreement (DUA). Approved access is typically granted
within 24 hours. The pipeline continues without them (OASIS rows are optional).

| Dataset | Source URL | Rows | Key Features | Access |
| --- | --- | --- | --- | --- |
| OASIS-1 (cross-sectional MRI) | `oasis-brains.org` | 416 | T1-MRI, MMSE, CDR, eTIV, nWBV, ASF | Sign DUA at oasis-brains.org |
| OASIS-2 (longitudinal MRI) | `oasis-brains.org` | 373 | Same + 3–4 visits per subject | Same DUA |

Once approved, download and place CSV files in `data/raw/oasis/`, then run:
```sh
python scripts/data/v5/process_oasis_v5.py
```

## Tier 3 — Programmatic APIs (no auth)

| Source | URL | What we pull | Script |
| --- | --- | --- | --- |
| OpenNeuro BIDS | `openneuro.org` | `participants.tsv` sidecars for neurological datasets | `scrape_openneuro.py` |
| gnomAD GraphQL | `gnomad.broadinstitute.org/api` | Variant frequencies for 15 neuro genes (APOE, LRRK2, HTT, SOD1, FUS, TARDBP, C9orf72, SCN1A) | `query_gnomad.py` |
| PubMed E-utilities | `eutils.ncbi.nlm.nih.gov/entrez/eutils` | 10,000 neuro abstracts for RAG corpus | `build_pubmed_corpus.py` |

**PubMed rate limits**: 3 req/s without an API key, 10 req/s with `NCBI_API_KEY`.
Register free at `ncbi.nlm.nih.gov/account/`.

## Synthetic augmentation (Tier —)

| Source | Rows | Purpose |
| --- | --- | --- |
| CTGAN (ALS augmentation) | ~650 | Rare class upsampling — ALS has <200 real rows |
| CTGAN (Huntington's augmentation) | ~648 | Rare class upsampling — HD has <100 real rows |

Synthetic rows are generated by `ctgan_augment.py` only when the real class count
falls below 200. Synthetic fraction is capped at 30% per class.
Rows are tagged `data_source = "ctgan_synthetic"` for drift-detection exclusion.

## Merged dataset

All sources are unified by `merge_v5.py` into `data/real_v5.parquet`:

```
56 features × ~16,000–20,000 rows
Label: DiseaseType (6-class) + risk_label (binary)
```

**Unified feature schema** (56 columns):

| Group | Features |
| --- | --- |
| Core clinical (26) | Age, MMSE, FunctionalAssessment, ADL, MemoryComplaints, BehavioralProblems, Depression, Hypertension, CardiovascularDisease, Diabetes, HeadInjury, PhysicalActivity, DietQuality, SleepQuality, CholesterolTotal/LDL/HDL/Triglycerides, SystolicBP, DiastolicBP, BMI, Smoking, AlcoholConsumption, FamilyHistoryAlzheimers, APOE4_dosage |
| Imaging-derived (6) | eTIV, nWBV, ASF, MR_Delay, hippocampus_volume, entorhinal_thickness |
| Biomarkers (6) | CSF_Abeta42, CSF_pTau, CSF_tTau, UPDRS_motor, UPDRS_total, nfl_plasma |
| Wearable/motion (6) | tremor_amplitude, gait_velocity, step_asymmetry, actigraphy_activity_index, HR_variability, SpO2_mean |
| Genomic (4) | APOE_risk_score, LRRK2_variant_freq, HTT_repeat_est, polygenetic_risk_score |
| Metadata (8) | data_source, dataset_tier, subject_id, DiseaseType, risk_label, age_group, sex, synthetic_flag |

Run the full pipeline:
```sh
PYTHONPATH=.:src python scripts/data/v5/merge_v5.py
```

Or via DVC (recommended for reproducibility):
```sh
dvc repro merge_v5
```
